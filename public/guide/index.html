      2、Python 数据科学

      Numpy
      Python 科学计算的基本包；强大的 N 维数组；数值计算工具。

      SHAP
      一种博弈论方法，用于解释任何机器学习模型的输出。

      Vega-Altair
      一个基于 Vega 和 Vega-Lite 的 Python 声明式统计可视化库。

      pandas
      一个快速、强大、灵活且易于使用的开源数据分析和操作工具。

      Matplotlib
      用 Python 创建静态、动画和交互式可视化。

      Streamlit
      一个开源 Python 框架；快速构建和部署强大的数据应用程序。

      Statsmodels
      Python 统计建模库；用于进行统计测试和统计数据探索。

      Seaborn
      一个基于 matplotlib 的 Python 数据可视化库。

      Folium
      轻松地在交互式传单地图上可视化使用 Python 处理的数据。

      Sklearn
      Python 中的机器学习；简单有效的预测数据分析工具。

      Plotly
      Plotly 的 Python 图形库可制作交互式、出版质量的图形。

      GeoPandas
      旨在让使用 Python 处理地理空间数据变得更加容易。
